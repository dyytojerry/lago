# Lago 产品官网

基于 Next.js 14 和 Tailwind CSS 的产品官网静态页面。

## 技术栈

- **Next.js 14**: React 全栈框架
- **TypeScript**: 类型安全
- **Tailwind CSS**: 原子化 CSS 框架
- **Framer Motion**: 动画库（滚动触发动画）

## 设计系统

遵循 Lago 设计系统规范：

- **主色**: 科技信赖蓝 `#00C4CC`
- **强调色**: 活力橙 `#FF8C69`
- **背景色**: 极浅灰 `#F8F8F8`
- **容器色**: 纯白 `#FFFFFF`
- **文本色**: `#2A2A2A` (标题/正文) / `#888888` (辅助文字)

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 页面结构

官网包含以下模块（基于产品白皮书）：

1. **Hero Section** - 产品定位和愿景
2. **产品定位** - 产品定位和愿景说明
3. **市场机会** - 痛点分析和解决方案
4. **AI 驱动** - AI 功能模块展示
5. **物业合作机制** - 合作模式和线下闭环
6. **独特竞争壁垒** - 核心竞争优势
7. **发展路径** - 产品 Roadmap
8. **结语** - 产品总结

## 特性

- ✅ 滚动触发的模块进场动画（使用 Framer Motion）
- ✅ 响应式设计（移动端和桌面端）
- ✅ 符合设计系统规范
- ✅ 平滑滚动导航
