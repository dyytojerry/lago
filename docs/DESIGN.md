# 📘 Lago - 来购（社区二手租售平台）

这套设计系统将深度融合 Lago 的 **社区信任、AI 驱动** 和 **租售乐趣** 三大定位。

---

# **✨ Lago 应用主题设计系统：Smart Circulation 2.0 (智能循环)**

**核心目标：** 在移动端（微信小程序）打造一个**高效、值得信赖**且**充满邻里温情**的二手/租赁交易体验。

## I. 主题风格与核心原则

| 风格维度               | 描述                            | 设计支撑（如何实现）                                 |
| :--------------------- | :------------------------------ | :--------------------------------------------------- |
| **可信赖 (Trust)**     | 专业的形象，建立交易信心。      | `科技信赖蓝`主色、小区认证徽章、清晰的信息架构。     |
| **乐趣性 (Playful)**   | 体现玩具/游戏的活力，鼓励参与。 | `活力橙`强调色、大圆角、富有表现力的图标。           |
| **AI 驱动 (Smart)**    | 突出 AI 带来的效率和个性化。    | 浅蓝色的 AI 提示条、GeoHash 抽象纹理、语义搜索界面。 |
| **高效性 (Efficient)** | 快速完成 C2C 和租赁交易。       | 简洁的表单、明确的 CTA、优化的交易链路。             |

## III. 字体与排版 (Typography & Hierarchy)

- **字体选择：** 移动端优先使用系统默认字体（如苹方 SC/San Francisco），保持性能和原生体验。
- **字重偏好：** 强调 **Semi-Bold (600)** 和 **Bold (700)** 在关键信息（商品价格、小区名称）上的使用，以提升信任感和信息获取效率。

| 元素                 | 尺寸 (px) | 字重 (Weight) | 颜色      | 示例应用                                   |
| :------------------- | :-------- | :------------ | :-------- | :----------------------------------------- |
| **页面标题 (Title)** | 24        | Bold          | `#2A2A2A` | 页面顶部的导航标题（如“商品列表”）。       |
| **模块标题 (H2)**    | 18        | Semi-Bold     | `#2A2A2A` | 列表中的分类标题、详情页的 Section Title。 |
| **商品价格**         | 20        | Bold          | `#FF8C69` | **突出显示，使用强调色。**                 |
| **商品名称**         | 16        | Semi-Bold     | `#2A2A2A` | 列表中的商品名称。                         |
| **辅助信息**         | 12        | Regular       | `#888888` | 发布时间、小区地理位置、虚拟号提示。       |
| **CTA Text**         | 16        | Semi-Bold     | `White`   | 按钮文字，确保清晰。                       |

## IV. 基础组件规范 (Core Components Specification)

### 1. 容器与圆角 (Containers & Corner Radius)

- **通用圆角 (Global Radius):** **`8px`** 或 **`10px`**。应用于所有卡片、按钮和输入框，保持友好和统一的“社区”感觉。
- **卡片阴影 (Shadow):** 统一使用 **轻微、柔和的底部阴影** (`0px 2px 4px rgba(0, 0, 0, 0.04)`)，将内容与背景分离，但不显得沉重。

### 2. 按钮系统 (Button System)

| 类型             | 颜色                      | 用途                                 | 形状             |
| :--------------- | :------------------------ | :----------------------------------- | :--------------- |
| **主要交易 CTA** | 活力橙填充                | 购买、租赁、立即发布、确认支付。     | 高圆角或胶囊形。 |
| **次要连接 CTA** | 科技信赖蓝填充            | 聊天、关注、加入小区频道、申请认证。 | 高圆角或胶囊形。 |
| **轻量级操作**   | 透明背景 + 科技信赖蓝边框 | 取消、返回、查看更多。               | 保持圆角。       |

### 3. 表单与输入 (Forms & Inputs)

- **登录/注册：** 简洁、大尺寸的输入框。使用**手机号 + 验证码**流程，强调快速、高效。
  - **焦点状态：** 输入框边框在获得焦点时使用 **1px 科技信赖蓝** 边框。
- **商品发布：** 使用清晰的 **分步导航 (Stepper)**，引导用户完成拍照、描述、定价等。使用 AI 模块辅助**自动识别商品、生成描述**。

## V. 核心功能组件设计 (Feature-Specific Components)

### 1. 商品列表页 (List View)

- **布局：** 瀑布流或双列卡片。每张卡片必须包含：
  1.  商品图片 (大图，圆角)。
  2.  **价格 (活力橙，Bold)。**
  3.  商品名称 (Semi-Bold)。
  4.  **地理位置/小区名称 (12px 辅助信息，Icon 科技信赖蓝)。**
- **筛选与搜索：**
  - **搜索框：** 默认提示语为“我想租个 Switch 一个周末” (语义搜索示例)，突出 AI 功能。
  - **地理筛选：** 在顶部导航栏明确显示“当前小区：[小区名称]”，且名称使用**`科技信赖蓝`**，强调小区定位。

### 2. 详情页 (Detail View)

- **信任模块：** 在卖家头像旁边，必须醒目展示 **“邻里认证”徽章**（科技信赖蓝）。
- **交易操作区：**
  - **租赁/购买 CTA：** 底部悬浮栏，使用 **`活力橙`** 按钮，清晰显示价格和操作。
  - **面交/自提提示：** 在商品描述区，使用一个**浅科技信赖蓝背景**的小卡片，突出“建议邻里面交，即时交易”。

### 3. AI 交互与提示 (AI Interactions)

- **智能推荐 Banner：** 在商品列表顶部，使用一个**浅科技信赖蓝**背景的 Banner，显示“基于您的邻里画像，AI 推荐了以下商品”。
- **智能客服入口：** 在个人中心或帮助页，AI 客服的对话气泡使用**`科技信赖蓝`**边框，人工客服使用中性灰，区分服务层级。

### 4. 社区与直播 (Community & Live)

- **小区频道 Tab：** Tab Bar 中的“小区频道”图标设计为一个融合了 **家和播放按钮** 的复合图标，使用`科技信赖蓝`激活色。
- **直播卡片：** 实时直播的入口卡片，使用 **半透明的`活力橙`** 角标提示“Live”或“活动中”，吸引用户。

---

## VI. 图标系统 (Iconography System)

- **风格：** 统一采用 **圆润的线描风格 (Rounded Line Icons)**，确保友好和现代感。
- **核心图标设计：**
  - **信任/安全：** 锁、盾牌、身份认证 → 使用 **`科技信赖蓝`**。
  - **乐趣/活动：** 玩具、游戏手柄、礼品、抽奖 → 使用 **`活力橙`**。
  - **交易/物流：** 购物车、租金、柜子 → 使用 **中性灰** 或 **`科技信赖蓝`**。
  - **AI/数据：** 芯片、节点、搜索 → 必须使用 **`科技信赖蓝`**，体现核心壁垒。

这套设计系统规范了 Lago 应用从底层色彩、排版到特色功能组件的所有视觉元素，确保了应用在实现所有功能时，都能保持主题一致性，并强化了 **“邻里信任”** 和 **“AI 驱动”** 的品牌定位。
