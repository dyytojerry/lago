## 一、产品目标

Lago 旨在构建一个以「小区为中心」的社区化二手租售与租赁平台，让邻里间可以基于信任完成 **玩具、游戏机等高频低用闲置品的租售循环**。
产品首期聚焦两类：

- 🎮 游戏机租赁/二手转卖
- 🧸 玩具租赁/置换

---

## 二、核心用户路径

### 用户端主要路径

```
首页 → 搜索/浏览商品 → 商品详情 → 聊天/虚拟号 → 下单或面交 → 评价/信用积累
```

### 商家端主要路径

```
商家中心 → 发布商品 → 管理库存与租期 → 订单/聊天 → 结算收入
```

### 小区物业端主要路径

```
物业管理端 → 管理循环柜 → 直播活动 → 提交活动数据与合作收益
```

## 四、页面功能明细

### 1️⃣ 首页（Home）

- 小区识别（基于微信地理定位）
- 推荐商品（租赁/买卖）
- 小区频道入口（直播、活动预告）
- 活动 Banner（物业合作推广）
- 搜索栏（模糊搜索 + 意图识别）

### 2️⃣ 搜索页（Search）

- 搜索框（支持自然语言输入）
- 筛选条件（价格、距离、类目、交易类型）
- 排序（最新上架、最近距离、租金最低）
- 搜索建议（AI 自动补全）

### 3️⃣ 商品详情页（Product Detail）

- 商品图片轮播、描述、价格
- 卖家身份：个人认证 / 小区住户 / 平台商户
- 租赁周期与押金规则
- 联系方式（聊天 / 虚拟号）
- 商品评论区 + 评分系统
- 举报与收藏功能

### 4️⃣ 聊天页（Chat）

- 支持商品卡片嵌入聊天气泡
- 虚拟号接入（通话按钮）
- 系统提示（订单状态、退款信息）
- 表情 / 图片发送
- 消息加密与防诈骗提示

### 5️⃣ 发布页（Publish）

- 选择类目（玩具 / 游戏机）
- 交易类型（出售 / 租赁）
- 商品描述、图片上传
- 价格、租期、押金、库存设置
- 发布预览 → 确认发布
- 审核机制（平台预审或 AI 检测）

### 6️⃣ 我的页面（Profile）

- 我的订单（待支付 / 租赁中 / 已完成）
- 我的收藏 / 发布记录
- 我的积分 / 信用等级
- 小区切换入口（可加入多个小区）
- 退出登录
