## 🎨 网站设计稿 - 视觉描述（Design Specs）

### 页面一：首页 - 愿景与信任闭环

#### **视觉主题：** 温暖的邻里感 + 科技的精准感。

#### **布局：** 标准 Web Layout (1440px 宽度)。

| 区域          | 视觉元素与布局描述                                                                                                                                                                                                                                                                                                                                                                |
| :------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Header**    | **Logo：** 左上角，Lago/来购文字，使用`#00C4CC`（科技信赖蓝）。**导航：** 居中对齐，使用圆润无衬线字体。                                                                                                                                                                                                                                                                          |
| **Hero 区**   | **布局：** 左右分栏 (60% 左侧文字 / 40% 右侧视觉)。**左侧 (文字)：** Headline (`#2A2A2A`深灰)，“零距离”和“租售”使用`#FF8C69`（活力橙）粗体高亮。**右侧 (视觉)：** 视觉主图是一张温暖的插画：一个现代风格的小区环境，一位家长正在与邻居交换一个玩具。背景颜色为浅青色渐变。**下方按钮：** 两个按钮，主 CTA 使用`活力橙`填充，圆角极高（Pill Shape），辅 CTA 使用`科技信赖蓝`边框。 |
| **价值模块**  | 三个圆角卡片，背景为柔和的浅`科技信赖蓝`（接近白色，带 5% 蓝色透明度）。每个卡片顶部有圆润、简洁的 **线描图标**（Iconography）。图标的填充色使用`活力橙`。底部为简短文字描述。卡片间有微小的阴影提升层次感。                                                                                                                                                                      |
| **Trust Bar** | 在价值模块下方，一排信任 Logo：**微信小程序、物业合作 (抽象图标)、AI 赋能 (抽象图标)。** 使用浅灰色调以保持首页的简洁。                                                                                                                                                                                                                                                           |

### 页面二：智能核心 - AI 驱动的服务中枢

#### **视觉主题：** 沉浸式科技感，突出数据与连接。

#### **布局：** 垂直滚动，模块化设计。

| 区域             | 视觉元素与布局描述                                                                                                                                                                                                                                                                                                                               |
| :--------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Section 背景** | 整个区域背景色为 **深海军蓝/墨绿** (`#1A2A3A`)，营造科技和夜间数据流动的氛围。                                                                                                                                                                                                                                                                   |
| **AI 核心模块**  | **标题：** 居中白色无衬线体。**内容：** 以 **抽象的 AI 节点图/数据流** 为视觉中心。这是一张由细小的白色或`科技信赖蓝`线条连接的抽象网络图。**功能卡片 (3 个)：** 以居中对齐的方式堆叠。卡片背景使用半透明的深色（或磨砂玻璃效果），文字白色。**高亮元素：** 功能名称（如“智能推荐”）使用`活力橙`强调色，配合一个代表 AI 的`科技信赖蓝`圆形图标。 |
| **技术壁垒区**   | **布局：** 一个中央展示区，显示一个简洁的流程图（或代码片段）。**流程图：** 使用粗线条的`科技信赖蓝`表示连接，节点使用`活力橙`圆点。**代码片段：** 模拟 VS Code 或终端界面的风格，浅蓝或绿色高亮关键技术栈名称。                                                                                                                                 |
| **CTA**          | 居中，一个“联系我们，探讨 AI 集成”的按钮，使用纯白色背景，**`科技信赖蓝`** 边框。                                                                                                                                                                                                                                                                |

### 页面三：社区闭环 - 物业合作与线下场景

#### **视觉主题：** 实体与虚拟的融合，强调 O2O 闭环。

#### **布局：** 分区展示，强调实体资产。

| 区域             | 视觉元素与布局描述                                                                                                                                                                                                                                                                                       |
| :--------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Section 标题** | 使用`科技信赖蓝`粗体标题。                                                                                                                                                                                                                                                                               |
| **智能柜模块**   | **主视觉：** 居中放置一个高精度的 **“玩具循环柜”** 渲染图。柜体是现代的白色/浅灰色，操作屏幕和 Logo 使用醒目的`科技信赖蓝`。**两侧文字：** 简洁的图标（使用`活力橙`）搭配文字，介绍智能柜的 3 个核心优势。**视觉连线：** 从智能柜图像引出一条条细线，连接到文字描述，强调柜子的数据同步能力。            |
| **活动直播模块** | **布局：** 左右分栏。**左侧（线下）：** 一张高质量照片或插画，展示充满笑容的社区活动（孩子玩玩具）。背景用`活力橙`的色块或装饰性几何图案点缀，强调“乐趣”和“活动”。**右侧（线上）：** 一个模拟手机界面 Mockup，显示小程序内的“小区频道直播”，直播界面元素（如评论、送礼）使用`科技信赖蓝`和`活力橙`高亮。 |
| **合作机制**     | **数据表格：** 使用简洁的黑白灰表格，但在“收益来源”和“抽佣比例”等关键数字上，使用**`科技信赖蓝`填充的小圆点** 或**粗体字**突出，提升专业感和可读性。                                                                                                                                                     |
| **Footer CTA**   | 底部区域使用浅灰色背景，一个醒目的`活力橙`大按钮，文案为“申请成为首批社区合作节点”。                                                                                                                                                                                                                     |
